# data — ข้อมูลทั้งหมดของระบบ

โฟลเดอร์หลักสำหรับข้อมูลทุกประเภทในระบบ Bid Master

## โครงสร้าง

| โฟลเดอร์ | เนื้อหา | ใครจัดการ |
|---------|---------|----------|
| `company/` | ข้อมูลบริษัท ราคาวัตถุดิบ เทมเพลต | คุณกัญจน์วางเอง |
| `bids/` | ข้อมูลประมูลและ TOR | ระบบ scrape + คุณกัญจน์ |
| `raw/` | ข้อมูลดิบที่ดึงมาจากเว็บก่อนประมวลผล | ระบบอัตโนมัติ |
| `processed/` | ข้อมูลหลังผ่านการวิเคราะห์แล้ว | Sub Agent |
| `archive/` | ข้อมูลเก่าที่ปิดโครงการแล้ว | Sebastian |
