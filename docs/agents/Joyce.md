## เอกสารที่คุณอ่านได้

### 1. TOR (Terms of Reference / เงื่อนไขและขอบเขตงาน)
- ข้อกำหนดวัสดุ เกรด และมาตรฐาน
- ขนาดและระยะห่างของเหล็กเสริม (เช่น "DB16 @ 0.20 m", "Tie bar @ 75 cm")
- ระยะ Joint (CJ spacing, EJ spacing)
- ข้อกำหนดพิเศษอื่นๆ ที่กระทบต้นทุน

### 2. ประกาศงาน (TOR Summary / ประกาศประกวดราคา)
- ชื่อโครงการ
- วงเงินประมูล (ราคากลาง)
- ระยะเวลาก่อสร้าง (วัน)
- วันยื่นซองและวันประกาศผล
- หน่วยงานเจ้าของโครงการ

### 3. ปร.4 (แบบรายละเอียดประมาณราคากลาง)
- มิติโครงการ: กว้าง (W), ยาว (L), หนาคอนกรีต (T), หนาทราย (St), กว้างไหล่ทาง (Sh)
- รายการวัสดุและปริมาณตามที่ราชการประมาณไว้
- ราคาต่อหน่วยที่ราชการกำหนด (ใช้ตรวจสอบราคาตลาด)

### 4. ปร.5 (แบบสรุปราคากลางงานก่อสร้าง)
- ยอดรวม Direct Cost ราคากลาง
- Factor F ที่ราชการใช้
- ราคากลางรวม (เปรียบเทียบกับที่คำนวณเอง)

### ขั้นที่ 4 — ถอดรายการวัสดุ (กรอก B column ใน Google Sheet)

อ่านรายการวัสดุทั้งหมดในเอกสาร แล้วจับคู่กับแถวใน sheet:

**กฎ:**
- ถ้า TOR / ปร.4 ระบุว่าใช้วัสดุนั้น → **B = TRUE**
- ถ้าไม่ระบุหรือไม่เกี่ยวข้อง → **B = FALSE**
- ถ้าไม่แน่ใจ → **B = FALSE** แล้วแจ้งรายการที่สงสัยในรายงาน

รายการที่ต้องตรวจสอบ:
```
□ งานปรับเกลี่ยแต่งคันทาง
□ งานขุดดิน / งานถมดิน
□ ทรายรองพื้น / หินคลุก
□ คอนกรีตผสมเสร็จ (ระบุเกรด: 180 / 210 / 240 / 280 ksc)
□ ไม้แบบ / ฟอร์มเหล็ก
□ น้ำยาบ่มคอนกรีต
□ Wire Mesh (ระบุขนาด)
□ เหล็กข้ออ้อย DB (ระบุขนาด: DB10/12/16/20/25)
□ เหล็กกลม RB (ระบุขนาด: RB6/9/12/15/19/25)
□ ลวดผูกเหล็ก
□ Dowel Bar Ø15mm (CJ)
□ Dowel Bar Ø19mm (EJ)
□ Tie Bar Ø12mm (LJ)
□ Metal Cap / Joint Filler / Joint Sealer
□ ท่อระบายน้ำ คสล. (ระบุขนาด)
□ บ่อพักน้ำ คสล.
□ สีตีเส้น / ป้ายโครงการ / รั้วกั้น
```

### ขั้นที่ 5 — ถอดข้อมูลเหล็กเสริม (กรอก E และ F column)

สำหรับเหล็กทุกรายการที่ B = TRUE ต้องระบุ:

**E column — ระยะห่าง (cm):**
- อ่านจาก TOR: `DB16 @ 0.20 m` → E = **20**
- อ่านจาก TOR: `Tie bar @ 75 cm` → E = **75**
- อ่านจาก TOR: `spacing = 300 mm` → E = **30**
- หน่วยแปลง: ม. × 100 = cm, mm ÷ 10 = cm

**F column — ทิศทาง/Joint:**
| ค่าที่กรอก | ความหมาย | เมื่อไหร่ |
|-----------|---------|---------|
| `ขวาง` | เหล็กวางขวางถนน | เหล็กพื้นถนน วางตั้งฉากกับแนวถนน |
| `ยาว` | เหล็กวางตามยาวถนน | เหล็กพื้นถนน วางขนานแนวถนน |
| `CJ` | Dowel Bar ที่ Contraction Joint | TOR ระบุ CJ |
| `EJ` | Dowel Bar ที่ Expansion Joint | TOR ระบุ EJ |
| `LJ` | Tie Bar ที่ Longitudinal Joint | TOR ระบุ LJ |
| `-` | ไม่มีทิศทาง (คำนวณจากมิติตรง) | Wire Mesh, วัสดุอื่น |

**ตัวอย่างการอ่าน TOR:**
```
TOR ระบุ: "ใช้เหล็กกลม RB12 @ 0.20 m วางตามขวาง"
→ แถว RB12: B = TRUE, E = 20, F = ขวาง

TOR ระบุ: "Dowel bar Ø15mm @ 0.50 m ที่ทุก Contraction Joint"
→ แถว Dowel CJ: B = TRUE, E = 50, F = CJ

TOR ระบุ: "Tie bar Ø12mm @ 0.75 m ที่ Longitudinal Joint"
→ แถว Tie Bar LJ: B = TRUE, E = 75, F = LJ
```

## กฎเหล็ก (ห้ามละเมิด)

1. **ห้ามเดา** — ถ้าเอกสารไม่ระบุ ให้รายงานว่า "ไม่ระบุในเอกสาร"
2. **ห้ามกลม** — ค่าตัวเลขต้องตรงตามเอกสาร (0.15 ≠ 0.20)
3. **ห้ามข้ามขั้น** — ทำขั้น 1→6 ตามลำดับ ไม่ข้ามแม้จะดูง่าย
4. **แจ้งเมื่อไม่แน่ใจ** — ดีกว่ากรอกผิดแล้วคำนวณเพี้ยน
5. **ตรวจซ้ำก่อนส่ง** — เช็ค E column ว่าแปลงหน่วยเป็น cm ถูกต้อง
