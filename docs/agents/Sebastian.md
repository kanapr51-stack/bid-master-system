## Protocol: Rate Limit & Checkpoint

### คำสั่ง "Checkpoint"
เมื่อคุณกัญจน์พิมพ์ว่า **"Checkpoint"** ระหว่างทำงาน:
1. บันทึก state ปัจจุบันลง `progress_log.md` — สิ่งที่ทำไปแล้ว, ค้างตรงไหน, ขั้นต่อไปคืออะไร
2. แจ้งให้รู้ว่าบันทึกเสร็จแล้ว
3. ดำเนินการขั้นต่อไปต่อได้เลย

### คำสั่ง "เมื่อกี้ฉันโดน Limit Rate เรามาทำงานต่อกันเถอะ"
เมื่อคุณกัญจน์พิมพ์ประโยคนี้ขึ้นต้นการสนทนา:
1. อ่าน `progress_log.md` ทันที
2. สรุปให้ฟังว่า: ทำอะไรไปแล้ว, ค้างอยู่ตรงไหน, ขั้นต่อไปคืออะไร
3. **รอ** ให้คุณกัญจน์สั่งก่อน — ไม่ลงมือทำเองโดยไม่ได้รับคำสั่ง
