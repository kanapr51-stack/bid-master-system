# cost data — ต้นทุนและวัตถุดิบ

โฟลเดอร์นี้เก็บวิธีการคำนวณต้นทุนและปริมาณวัตถุดิบที่ต้องใช้ในแต่ละประเภทงานของบริษัท

---

## ไฟล์ในโฟลเดอร์นี้

| ไฟล์ | เนื้อหา |
|-----|---------|
| `road_construction.md` | วิธีคำนวณต้นทุนและวัตถุดิบงานสร้างถนน |
| `concrete.md` | วิธีคำนวณต้นทุนและวัตถุดิบคอนกรีตผสมเสร็จ |
| `general_construction.md` | วิธีคำนวณต้นทุนงานก่อสร้างทั่วไป |

---

## หมายเหตุ

- แต่ละไฟล์เก็บสูตรคำนวณ, หน่วยวัด, และอัตราส่วนวัตถุดิบ
- ใช้ข้อมูลในโฟลเดอร์นี้ประกอบการสร้างระบบ Sub Agent B (คำนวณต้นทุน)
- อัปเดตเมื่อราคาหรือสูตรมาตรฐานเปลี่ยนแปลง
