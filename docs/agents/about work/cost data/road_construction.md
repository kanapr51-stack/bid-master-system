# งานสร้างถนน คสล. — สูตรคำนวณปริมาณวัสดุและต้นทุน

---

## ตัวแปรนำเข้า (Input Variables)

| สัญลักษณ์ | ความหมาย               | หน่วย | ค่าปกติ   |
| --------- | ---------------------- | ----- | --------- |
| W         | กว้างถนน               | ม.    | 4.00      |
| L         | ยาว                    | ม.    | -         |
| T         | หนาคอนกรีต             | ม.    | 0.15-0.20 |
| St        | หนาทรายรองพื้น         | ม.    | 0.05-0.10 |
| Sh        | กว้างไหล่ทาง (ข้างละ)  | ม.    | 0.50      |
| CJ        | ระยะ Contraction Joint | ม.    | 5-10      |
| EJ        | ระยะ Expansion Joint   | ม.    | 100       |

## สูตรคำนวณปริมาณวัสดุ (ถอดจากมาตรฐานราชการ)

### 1. งานดิน
```
งานปรับเกลี่ยแต่งคันทางเดิม (ตร.ม.) = (W + Sh × 2) × L
งานขุดดิน / ถมดิน (ลบ.ม.)            = ดูแบบ
```

### 2. ทรายรองพื้น
```
ปริมาณทรายรองพื้น (ลบ.ม.) = W × L × St
```

### 3. คอนกรีต
```
ปริมาณงานคอนกรีต (ลบ.ม.) = W × L × T
พื้นที่ผิวคอนกรีต (ตร.ม.)  = W × L
```

### 4. Wire Mesh
```
กรณีใช้ Wire Mesh RB4 @ 0.20×0.20:
  ปริมาณ (ตร.ม.) = W × L  (คิดต่อแผง: W_panel × L_panel)
```

### 5. Dowel Bar — Contraction Joint (CJ)
```
จำนวน CJ = FLOOR(L/CJ, 1) - 1 - จำนวน EJ
ความยาวรวม CJ = W × จำนวน CJ
จำนวนเหล็กต่อ CJ = FLOOR(W / 0.50, 1)  [@ spacing 0.50 ม.]
ความยาวเหล็กต่อท่อน = 0.50 ม.
น้ำหนักรวม Dowel Bar ∅15mm (กก.) = จำนวนเหล็ก × 0.50 × จำนวน CJ × 1.390
  → หน่วยน้ำหนัก ∅15mm = 1.390 กก./ม.
```

### 6. Dowel Bar — Expansion Joint (EJ)
```
จำนวน EJ = FLOOR(L/EJ, 1) - 1
จำนวนเหล็กต่อ EJ = FLOOR(W / 0.50, 1)
ความยาวเหล็กต่อท่อน = 0.50 ม.
น้ำหนักรวม Dowel Bar ∅19mm (กก.) = จำนวนเหล็ก × 0.50 × จำนวน EJ × 2.230
  → หน่วยน้ำหนัก ∅19mm = 2.230 กก./ม.
Metal Cap (ชุด) = จำนวนเหล็กต่อ EJ × จำนวน EJ
```

### 7. Joint Filler & Sealer — Expansion Joint
```
Joint Filler พื้นที่ (ตร.ม.) = W × (T - 0.025) × จำนวน EJ
Joint Sealer (ลิตร)          = W × 0.025 × 0.025 × 1,000 × จำนวน EJ
  → ความกว้างร่อง = 0.025 ม., ความลึกร่อง = 0.025 ม.
```

### 8. Joint Sealer — Contraction Joint
```
Joint Sealer CJ (ลิตร) = W × 0.010 × 0.0375 × 1,000 × จำนวน CJ
  → ความกว้างร่อง = 0.010 ม., ความลึกร่อง = 0.0375 ม.
```

### 9. ไหล่ทาง (Shoulder)
```
ปริมาณงานไหล่ทาง (ลบ.ม.) = (T + St) × Sh × L × 2
```

## โครงสร้างต้นทุน (Thai Government Standard)

```
ราคาเสนอประมูล = Direct Cost × Factor F

โดย:
  Direct Cost = วัตถุดิบ + ค่าแรง + เครื่องจักร
  Factor F    = ค่าตัวเลขตามตาราง กรมบัญชีกลาง
               ครอบคลุม: overhead + กำไร + ดอกเบี้ยเงินกู้

Factor F ปัจจุบัน (ดอกเบี้ย 7%, ว499 ปี 2566):
  - งานก่อสร้างทั่วไป ดูตาราง F ตามมูลค่างาน
  - ยิ่งงานมูลค่าน้อย F ยิ่งสูง (overhead % มากกว่า)
```

## ตัวอย่างคำนวณ (ถนน W=4m, L=40m, T=0.15m, St=0.05m, Sh=0.5m, CJ=10m, EJ=100m)

| วัสดุ | สูตร | ผล | หน่วย |
|-----|-----|---|------|
| งานปรับเกลี่ย | (4+0.5×2)×40 | 200 | ตร.ม. |
| ทรายรองพื้น | 4×40×0.05 | 8 | ลบ.ม. |
| คอนกรีต | 4×40×0.15 | 24 | ลบ.ม. |
| Wire Mesh | 4×40 | 160 | ตร.ม. |
| จำนวน EJ | FLOOR(40/100)-1 | 0 | ช่วง |
| จำนวน CJ | FLOOR(40/10)-1-0 | 3 | ช่วง |
| Dowel Bar 15mm (CJ) | FLOOR(4/0.5)×0.5×3×1.390 | 16.68 | กก. |
| Joint Sealer CJ | 4×0.01×0.0375×1000×3 | 4.50 | ลิตร |
| งานไหล่ทาง | (0.15+0.05)×0.5×40×2 | 8 | ลบ.ม. |

## สูตร Excel/Google Sheets (อ้างอิง C11=W, C12=L, C13=T, C14=St, C15=Sh, C16=CJ, C17=EJ)

```excel
งานปรับเกลี่ย  = ($C$11 + $C$15*2) * $C$12
ทรายรองพื้น   = $C$11 * $C$12 * $C$14
คอนกรีต       = $C$11 * $C$12 * $C$13
Wire Mesh      = $C$11 * $C$12
จำนวน EJ      = MAX(0, FLOOR($C$12/$C$17, 1) - 1)
จำนวน CJ      = MAX(0, FLOOR($C$12/$C$16, 1) - 1 - จำนวน EJ)
Dowel ∅15mm   = FLOOR($C$11/0.5,1) * 0.5 * จำนวน CJ * 1.390
Dowel ∅19mm   = FLOOR($C$11/0.5,1) * 0.5 * จำนวน EJ * 2.230
Metal Cap      = FLOOR($C$11/0.5,1) * จำนวน EJ
Filler EJ      = $C$11 * ($C$13 - 0.025) * จำนวน EJ
Sealer EJ      = $C$11 * 0.025 * 0.025 * 1000 * จำนวน EJ
Sealer CJ      = $C$11 * 0.01 * 0.0375 * 1000 * จำนวน CJ
ไหล่ทาง       = ($C$13 + $C$14) * $C$15 * $C$12 * 2
```
