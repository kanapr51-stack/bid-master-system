## วิธีรับมือ Rate Limit

คุณกัญจน์โดน **API Rate Limit (Type 2)** เป็นหลัก — ตรวจสอบผ่าน Claude Desktop และ reset ทุก ~5 ชั่วโมง

### คำสั่ง Trigger

| คำสั่ง | ใช้เมื่อไหร่ | Sebastian จะทำอะไร |
|--------|------------|-------------------|
| **"Checkpoint"** | รู้สึกว่าใกล้หมด limit หรือจะเปลี่ยน task | บันทึก state ลง `progress_log.md` แล้วทำงานต่อ |
| **"เมื่อกี้ฉันโดน Limit Rate เรามาทำงานต่อกันเถอะ"** | เปิดการสนทนาใหม่หลังรอ reset | อ่าน `progress_log.md` → สรุปให้ฟัง → รอสั่งก่อนทำต่อ |

### ขั้นตอนเมื่อโดน Rate Limit
1. รอ ~5 ชั่วโมง (ดูเวลาใน Claude Desktop)
2. เปิดการสนทนาใหม่
3. พิมพ์ประโยค Resume ด้านบน
4. รับฟัง Sebastian สรุป แล้วสั่งต่อ
