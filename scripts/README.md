# scripts — สคริปต์อัตโนมัติ

โฟลเดอร์นี้เก็บโค้ด Python และสคริปต์ที่ Sebastian ใช้รันงานอัตโนมัติ

## สคริปต์ปัจจุบัน

| ไฟล์ | หน้าที่ | สถานะ |
|-----|---------|-------|
| `Sebastian Python Pull Data.py` | ดึงข้อมูลประมูลจากเว็บภาครัฐ | กำลังพัฒนา |
| `sheets_client.py` | เชื่อมต่อและเขียนข้อมูลลง Google Sheets | กำลังพัฒนา |

## สคริปต์ที่จะเพิ่มในอนาคต

| ไฟล์ | หน้าที่ |
|-----|---------|
| `tor_reader.py` | อ่านและวิเคราะห์ TOR (Sub Agent A) |
| `cost_calculator.py` | คำนวณต้นทุน (Sub Agent B) |
| `bid_ranker.py` | จัดอันดับโครงการ (Sub Agent C) |
| `price_updater.py` | อัปเดตราคาวัตถุดิบอัตโนมัติ (Sub Agent E) |
